# Images Directory

This directory is for storing images used in your text-to-speech research demo page.

## Recommended Images

1. **methodology-diagram.png** - A visual representation of your TTS architecture or methodology
2. **results-chart.png** - Visualization of your evaluation results (bar charts, line graphs, etc.)
3. **Your institution logo** (if applicable)

## Image Guidelines

- Use PNG format for diagrams and charts for best quality
- Recommended width: 800-1200px (will be scaled to fit container)
- Use SVG format if possible for diagrams (best scaling quality)
- Keep file sizes reasonable (<500KB per image if possible)

## Placeholder Images

The demo page has fallback placeholder images that will appear if the specified images are not found.
